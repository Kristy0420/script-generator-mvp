# 🎯 使用指南

## 第一次使用

### 1. 设置 API Key

在项目根目录的 `.env.local` 文件中配置你的 Claude API Key：

```bash
ANTHROPIC_API_KEY=sk-ant-xxxxxxxxxxxx
```

如果没有 API Key，请访问：https://console.anthropic.com/settings/keys

### 2. 启动项目

```bash
npm run dev
```

浏览器访问：http://localhost:3000

## 功能详解

### 1. 知识点信息配置

#### 知识点名称（必填）
- **作用**：核心主题，会在剧本中反复强调
- **建议**：简洁明确，5-15 个字
- **示例**：
  - ✅ `如何制作咖啡`
  - ✅ `JavaScript 闭包`
  - ❌ `关于咖啡的一些知识点的讲解` （太冗长）

#### 详细描述（可选）
- **作用**：为 AI 提供更多背景信息，生成更准确的内容
- **建议**：2-3 句话，说明要讲什么、为什么重要
- **示例**：
  ```
  介绍手冲咖啡的完整流程，包括器具准备、水温控制、
  冲泡手法等。帮助初学者在家也能做出专业级别的咖啡。
  ```

#### 关键词（可选）
- **作用**：AI 会围绕这些关键词展开对话
- **格式**：用英文或中文逗号分隔
- **建议**：3-8 个关键词
- **示例**：`咖啡豆, 研磨, 水温, 手冲壶, 滤纸`

#### 背景信息（可选）
- **作用**：补充上下文，让对话更有深度
- **建议**：可以包括历史、趋势、争议点等
- **示例**：
  ```
  手冲咖啡起源于日本，近年在精品咖啡文化中流行。
  相比意式咖啡，手冲更能体现咖啡豆的原味特性。
  ```

---

### 2. 配置参数

#### 难度级别（必填）
- **作用**：影响对话的专业程度和术语使用
- **支持自定义**：可以输入任何标签
- **常用选项**：
  - `入门` / `初级` - 适合零基础受众
  - `中级` / `进阶` - 适合有一定了解的受众
  - `高级` / `专业` / `专家` - 适合专业人士

#### 复现次数（1-10）
- **作用**：知识点在剧本中出现的次数
- **建议**：
  - 简单概念：2-3 次
  - 复杂概念：4-6 次
  - 需要强化记忆的：7-10 次
- **效果**：AI 会用不同角度、不同表达方式重复知识点

---

### 3. 角色设置

#### 主持人
- **典型角色**：播客主持、节目主播、访谈者
- **性格建议**：
  - 友善、专业、耐心
  - 善于提问、引导话题
  - 亲切、幽默、活泼

#### 嘉宾
- **典型角色**：专家、老师、从业者
- **性格建议**：
  - 专业、权威、严谨
  - 热情、善于举例、通俗易懂
  - 风趣、有故事、接地气

**搭配建议**：
- 主持人活泼 + 嘉宾严谨 = 轻松但专业
- 主持人专业 + 嘉宾热情 = 深度且有趣
- 两人都活泼 = 娱乐性强，适合轻松话题

## 使用技巧

### 1. 如何获得更好的剧本？

✅ **详细填写知识点信息**
- 描述、关键词、背景信息越详细，生成的内容越精准

✅ **合理设置复现次数**
- 简单概念不要设置太多次，避免重复
- 复杂概念适当增加次数，强化理解

✅ **明确角色性格**
- 性格设置会影响对话风格和语言表达

✅ **选择合适的难度**
- 根据目标受众选择难度级别
- 初学者内容避免过多专业术语

### 2. 生成不满意怎么办？

🔄 **点击"重新生成"**
- AI 每次生成的结果都不同
- 可以多次尝试，选择最满意的版本

📝 **调整输入参数**
- 修改描述或关键词
- 调整难度级别或复现次数
- 改变角色性格设置

### 3. 如何使用生成的剧本？

📋 **复制到剪贴板**
- 点击"复制剧本"按钮
- 粘贴到 Word、Notion 等工具继续编辑

💡 **查看知识点标记**
- 黄色高亮的对话包含知识点
- 确认知识点是否按预期次数出现

✂️ **二次加工**
- 复制后可以手动修改对话
- 添加开场白、结尾语、过渡语等

## 常见问题

### Q1: 生成很慢怎么办？
**A**: 
- 正常生成时间为 10-30 秒
- 如果超过 1 分钟，可能是网络问题或 API 限流
- 刷新页面重试

### Q2: 提示 API Key 错误？
**A**:
- 检查 `.env.local` 文件是否正确配置
- 确认 API Key 有效且有余额
- 重启开发服务器 `npm run dev`

### Q3: 生成的对话太短？
**A**:
- 增加复现次数
- 在"详细描述"中添加更多内容
- 在"背景信息"中补充更多素材

### Q4: 生成的内容不符合预期？
**A**:
- 检查知识点名称是否清晰
- 丰富关键词和描述信息
- 调整难度级别
- 多次重新生成

### Q5: 可以生成多人对话吗？
**A**:
- 当前版本只支持 2 人（主持人+嘉宾）
- 后续版本会支持更多角色

## 进阶用法

### 1. 批量生成

如果需要生成多个相关剧本：
1. 准备一个主题列表
2. 逐个输入生成
3. 复制保存到文档
4. 最后统一编辑整理

### 2. 系列内容制作

制作系列播客/视频：
- 第 1 集：难度=入门，复现=3
- 第 2 集：难度=中级，复现=4
- 第 3 集：难度=高级，复现=5

### 3. 不同风格尝试

同一个知识点，尝试不同角色性格：
- 版本 A：主持人严肃、嘉宾幽默
- 版本 B：主持人活泼、嘉宾严谨
- 版本 C：两人都轻松活泼

选择最符合品牌调性的版本。
