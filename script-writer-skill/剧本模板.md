# 剧本写作规范模板

## 一、基本结构

每个剧本片段（Part）包含以下固定元素：

### 1. 标题区（必填）

```markdown
# D{天数}P{片段号}

**标题**：{英文标题}

**角色**：{角色1}, {角色2}, {角色3}

**知识点**：{单词1}, {单词2}, -{句型问句} -{句型答句}

**梗概**：{剧情摘要，100-200字，描述主要冲突和结局}
```

### 2. 场景设定（可选）

用于复杂场景或道具的详细说明：

```markdown
【{场景/道具名称}设定：{详细描述}】
```

### 3. 场景分割（必填）

```markdown
**SCENE {序号}****{时间}，{内/外}，{场景名称}**
```

时间格式：日/夜
内外格式：内/外

---

## 二、内容格式

### 1. 镜头描述

普通文本描述画面和动作：
- 景别描述：大景别、全景、中景、近景、特写
- 动作描述：角色动作和表情
- 镜头运动：上摇、下摇、推、拉、跟

```markdown
大景别，游乐场空镜&Z尖叫画外音。

全景zack、Michael和Jenna在游戏机前

镜头向上推，最高处红色区域有一个巨型豪华礼包图标
```

### 2. 特殊镜头标注

```markdown
**特写**（{拍摄对象}）。{描述}

**转场**
```

### 3. 对话格式

```markdown
**{角色名}**: ({表情/动作}) {台词内容}
```

- 角色名加粗
- 括号内为表演指导
- 知识点单词/句型用 `**加粗**` 标注

示例：
```markdown
**Jenna**: (冷静) Step back, guys. Watch me **scream**.

**Michael**: (好奇地凑过来) **How are you feeling**, Jenna?

**Jenna**: (尴尬) **I feel fine**. I won! Yay...
```

### 4. 音频标记

```markdown
音：{编号}
```

用于标记需要配音或音效的位置。

### 5. 分镜图片

```markdown
![图片](图片路径或base64)
```

## 三、完整示例

```markdown
# D2P1

**标题**：The Super Screamer

**角色**：Jenna, Michael, Zack 

**知识点**：scream, dizzy, -How are you feeling? -I feel...

**梗概**：游乐场举行"尖叫赢大奖"活动，Zack 和 Michael 轮番挑战，喊到嗓子冒烟、筋疲力尽也没能打破纪录。就在 Jenna 慢条斯理准备展示"真正的技术"时，一只倒挂的蜘蛛突然落在话筒上。受惊的 Jenna 瞬间爆发出穿透耳膜的非人类尖叫，不仅震坏了分贝计，更创下了惊人的新纪录。

【街机屏幕设定：一个分贝仪游戏机，可以根据呐喊的声音大小进行游戏互动，屏幕上有一个彩虹能量条】

**SCENE 1****日，外，游乐场挑战区** 

大景别，游乐场空镜&Z尖叫画外音。

**Zack**: (大叫) AAAAAHHHHHH!

**特写**（分贝仪）。屏幕底部的能量条随着声音一格一格往上涨。

音：66

**Michael**: Let me try!

**Michael**: (尖叫) EEEEEKKKKK!

音：67

**SCENE 2****日，外，Jenna的挑战** 

**Jenna**: (冷静) Step back, guys. Watch me **scream**.

**Michael**: (好奇地凑过来) **How are you feeling**, Jenna?

**Jenna**: (尴尬) **I feel fine**. I won! Yay...
```

## 四、写作要点

1. **知识点融入**：每个剧本必须自然融入指定的知识点（单词和句型）
2. **角色一致性**：保持角色性格特点（如 Jenna 通常是聪明/冷静的，Zack 容易紧张等）
3. **场景连贯**：场景之间的转换要有逻辑性
4. **表演指导**：为每句台词提供清晰的表情/动作指导
5. **镜头语言**：使用专业的镜头描述术语
6. **反转/喜剧效果**：剧情要有出人意料的反转或喜剧效果
