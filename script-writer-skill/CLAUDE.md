# Trial Class 项目说明

## 剧本写作

本项目包含儿童英语教学剧本。

### 使用剧本生成器

使用 `/script-writer` 命令启动剧本生成器，按照规范模板创建新剧本。

### 剧本模板参考

详见 [剧本模板.md](剧本模板.md)

### 剧本文件命名规范

- 格式：`S{季}W{周}D{天}剧本.md`
- 示例：`S5W1D2剧本.md` = 第5季第1周第2天的剧本

### 剧本结构要点

1. 标题区：编号、标题、角色、知识点、梗概
2. 场景设定：道具和场景详细说明
3. 场景分割：`**SCENE X****时间，内/外，场景名**`
4. 对话格式：`**角色名**: (表情/动作) 台词`
5. 知识点用 `**加粗**` 标注
